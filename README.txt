Данный проект предоставляет потенциальному пользователю различные возможности по поиску, просмотру детских кружков и подачи заявки на них.

Функционал сайта предоставлен пользователю следующими критериями:
1. Возможность авторизации через базу данных (далее — БД);
2. Возможность регистрации через БД;
3. Возможность просмотра кружков, оставления заявкок на запись на них;
4. Возможность просмотра расписания каждого кружка;
5. Возможность оставления отзыва о любом детском кружке, при условии авторизованной сессии.

Функционал сайта предоставлен администратору следующими критериями:
1. Изменение, добавление, удаление кружков;
2. Изменение, добавление, удаление расписаний для кружков;
3. Одобрение и отвержение заявок.

---------------------------------------------------------

БД проекта расположена в zip-архиве с названием "prDB".

Диаграммы "to be" и "ER" расположены в zip-архиве с названием "Diagrams". В папке также имеются снимки экрана диаграм.

Необходимая конфигурация системы для корректной работы проекта:
- PHP: 8.1
- БД: MariaDB-10.8
